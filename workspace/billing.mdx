Each workspace is on its own **plan**, which determines the features available to it — including [programmatic API access](/workspace/api). The current plan is shown next to the workspace name in the switcher.

## Manage billing

Open the **workspace switcher** (top-left) and select **Workspace settings**, then go to **Billing** to view your current plan, update payment details, and change your subscription. Only the workspace **owner** can manage billing. See [Members and roles](/workspace/members).

</Frame>

## Changing plans

Upgrade when you need more capacity or features such as API access; downgrade or cancel from the same screen. Changes apply to the workspace, so every member benefits from an upgrade immediately.

<Note>
  Billing is per workspace. If you run separate workspaces — for example, one per client — each has its own plan and invoices.
</Note>
